🏁 FinnTrack Setup

Use your phone to track your boat during races

📱 Step 1: Install Traccar Client

Download and install the Traccar Client app on your smartphone:

  • iPhone: Download from App Store
  • Android: Download from Google Play

⚙️ Step 2: Configure Traccar Settings

Open the Traccar Client app and configure these settings:

Server Settings:

Server URL: https://api.finntracker.org/traccar
Device Identifier: [YOUR-BOAT-ID] (see Step 3)
Frequency: 10 seconds
Distance: 10 meters
Important: Make sure "Location access" is set to "Allow all the time" in your phone settings for the Traccar app.

🚤 Step 3: Choose Your Boat ID

Your Device Identifier should be your sail number or boat name. Examples:

  • AUS123 (for sail number AUS 123)
  • NZL456 (for sail number NZL 456)
  • GBR789 (for sail number GBR 789)
  • MyBoatName (if no sail number)
Note: Use your actual sail number or boat name. This will be displayed on the live tracking map.

🏆 Step 4: Race Codes (Optional)

By default, your tracking will go to the training race. To join a specific race, add the race code to your server URL:

Australian Nationals 2026

?race=AUSNATS-2026-R01
Race 1 through R06

Gold Cup 2026

?race=GOLDCUP-2026-R01
Race 1 through R10

World Masters 2026

?race=MASTERS-2026-R01
Race 1 through R08

Training/Practice

?race=TRAINING-2026-R01
Default if no race specified

Example with race code:

Server URL: https://api.finntracker.org/traccar?race=AUSNATS-2026-R01&key=finntrack_local_dev_9f3a8c2e4d7b1a0c6e5f
API Key Required: Add &key=finntrack_local_dev_9f3a8c2e4d7b1a0c6e5f to your server URL for authentication.

🟢 Step 5: Start Tracking

Once configured:

  1. Tap the "Status: Stopped" button to start tracking
  2. Status should change to "Status: Online"
  3. Your location will be sent every 10 seconds while sailing
  4. Check finntracker.org to see your boat on the live map
Success! Your boat should now appear on the live tracking website with a green indicator showing you're active.

🔧 Troubleshooting

  • Not appearing on map? Check your internet connection and GPS signal
  • Red/offline status? Make sure tracking is started and location permissions are enabled
  • Wrong race? Double-check your server URL includes the correct race code
  • Battery drain? Use a portable charger - GPS tracking uses battery power
